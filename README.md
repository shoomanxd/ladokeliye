# 🍫 Sorry Website

A simple apology page. Pure black screen → double tap → chocolate cake reveal.

## Files in this repo

```
index.html   ← the whole website
cake.png     ← the chocolate cake photo (required!)
vercel.json  ← Vercel deployment config
README.md    ← this file
```

> ⚠️ **cake.png must be in the repo** — the site won't show the cake without it.

## Deploy on Vercel

1. Push all files (including `cake.png`) to a GitHub repo
2. Go to [vercel.com](https://vercel.com) → **Add New Project** → import your repo
3. Click **Deploy** — done!

Your live link will look like: `https://your-repo-name.vercel.app`

### Or via CLI
```bash
npm i -g vercel
vercel
```
