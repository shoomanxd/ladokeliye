# 🎂 Sorry Website

A sweet surprise webpage to say sorry to someone special. Built with pure HTML/CSS/JS — no dependencies, no build step.

## How it works

1. Opens on a dark chocolate-themed screen saying **"Double Touch For Surprise"**
2. On double-tap (or double-click on desktop), reveals a **giant chocolate cake** with the message *"SORRY I SLEPT WITHOUT TELLIN YOU 😭"*
3. Fully mobile-friendly and works offline

## Deploy on Vercel

### Option 1 — Via Vercel Dashboard (easiest)
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) and sign in
3. Click **"Add New Project"** → Import your GitHub repo
4. Leave all settings as default → click **Deploy**
5. Your site will be live at `https://your-project.vercel.app` ✅

### Option 2 — Via Vercel CLI
```bash
npm i -g vercel
vercel
```
